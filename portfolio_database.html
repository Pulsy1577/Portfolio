Project Description

The goal of this project was to create a relational database in MySQL while learning and adhering to proper realtional database structure. The database was designed, normalized, then implimented from scratch, no tools such as Zoho or any other database design tools were used. The link above will take you to a site where you can interface with the database directly. It is a sports betting database so it allows you to place bets, check if placed bets are winners and cashing out placed bets to a users wallet. All functionality, as well as test inputs are outlined on the site.
